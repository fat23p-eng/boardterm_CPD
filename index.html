🔐 ขั้นตอน Google OAuth Login
1

สร้าง Google Cloud Project

ไปที่ console.cloud.google.com → New Project

2

เปิด Google Identity API

APIs & Services → Library → "Google Identity" → Enable

3

สร้าง OAuth Client ID

Credentials → Create → OAuth 2.0 Client ID → Web application
ใส่ Authorized Origins: URL ของ Netlify site

4

ใส่ Client ID ในระบบ

นำ Client ID มาใส่ในช่องด้านบน → บันทึก → Reload

📊 ขั้นตอน Google Sheets Real-time
1

สร้าง Google Sheet

สร้าง Sheet ชื่อ: สหกรณ์, กรรมการ, ผู้ใช้

2

วาง Apps Script Code

Extensions → Apps Script → วางโค้ด [คลิกดูโค้ด]

3

Deploy Web App

Deploy → New deployment → Type: Web App
Execute as: Me | Who has access: Anyone

4

ใส่ URL ในระบบ

คัดลอก Web App URL ใส่ในช่อง "Google Apps Script URL" ด้านบน

🚀 Deploy บน Netlify
# วิธีที่ 1: Drag & Drop
เปิด app.netlify.com → ลากโฟลเดอร์ coop-v3 มาวาง

# วิธีที่ 2: GitHub + Auto Deploy
git init && git add . && git commit -m "init"
git remote add origin https://github.com/YOUR/REPO.git
git push -u origin main
# จากนั้น Netlify → New site → Import from Git

# วิธีที่ 3: Netlify CLI
npx netlify-cli deploy --prod --dir=.
